# dotfiles

Maintained using [chezmoi](https://www.chezmoi.io/).

# Setup

- Install [kitty](https://sw.kovidgoyal.net/kitty/).
- Install [starship.rs](https://starship.rs/).
- Install [rust](https://www.rust-lang.org/), [cargo-binstall](https://github.com/cargo-bins/cargo-binstall)
  [fzf](https://github.com/junegunn/fzf), [ripgrep](https://github.com/BurntSushi/ripgrep), [fd](https://github.com/sharkdp/fd), [bat](https://github.com/sharkdp/bat),
  [delta](https://github.com/dandavison/delta), [gh](https://cli.github.com/), [gh-install](https://github.com/redraw/gh-install), [gitui](https://github.com/extrawurst/gitui),
  [yazi](https://github.com/sxyazi/yazi), zsh, fish, [inshellisense](https://github.com/microsoft/inshellisense) and [carapace](https://github.com/carapace-sh/carapace).
- Install Neovim using [bob](https://github.com/MordechaiHadad/bob)
- Install R (`r-base`/`r-base-dev`) and [RStudio](https://posit.co/download/rstudio-desktop/).
- Install [chezmoi](https://www.chezmoi.io/) and the dotfiles.
