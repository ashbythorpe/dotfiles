# dotfiles

# Setup

* Install [nushell](https://www.nushell.sh/)
* Install [starship.rs](https://starship.rs/)
* Set nushell as default shell (e.g. `sudo chsh -s ~/.cargo/bin/nu`)
* Install [Neovim](https://github.com/neovim/neovim/wiki/Installing-Neovim#install-from-download)
* Install R (`r-base`/`r-base-dev`) and [RStudio](https://posit.co/download/rstudio-desktop/)
